# NXP_LPC1768

LPC1768 is arm cortex-m3 micro controller with on chip 512kb flash and 32kb RAM etc..

In this repository bare metal coding was done to test differnt peripherals like:

#### GPIO
#### PWM
#### ADC
#### DAC
#### Interrupt 
#### SPI
#### I2C
#### CAN
#### UART
#### TIMER

Also few external peripherals like :

#### LCD
#### KEYPAD
#### GPS
#### GSM
#### RFID
#### Temperature sensor
